== Verified clean areas

* Java: no System.out/printStackTrace/slf4j/log4j in main sources; no hardcoded credentials in main code; no duplicate LogRecord identifiers; nifi-cuioss-api clean.
* Java tests: uniformly high quality (JUnit5, @DisplayName/@Nested everywhere, no Mockito/PowerMock, no @Disabled, no empty tests).
* JS: no raw console.log in production/e2e code; no `var`; const/let discipline good; element-content XSS handling solid; no skipped/focused tests.
* Docs: zero broken xref/link/include targets; module tables accurate; rest.gateway.* property docs match PropertyDescriptors 1:1.
* Baseline `mvn clean verify` green.
