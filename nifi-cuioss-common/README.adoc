= nifi-cuioss-common

Shared JWT infrastructure: `StandardJwtIssuerConfigService` implementation, configuration management (including `ConfigurationManager`, `IssuerConfigurationParser`, `IssuerPropertyDescriptorFactory`), and utilities (including `AuthorizationValidator`, `TokenClaimMapper`, `ErrorContext`, `TokenMasking`, and others).

See link:../doc/architecture/README.adoc[Architecture Overview] for module relationships and link:../doc/architecture/jwt-processor.adoc[JWT Processor Architecture] for the Controller Service lifecycle.
