= nifi-cuioss-common

Shared JWT infrastructure: `StandardJwtIssuerConfigService` implementation, configuration management (`ConfigurationManager`, `IssuerConfigurationParser`), and utilities (`AuthorizationValidator`, `TokenClaimMapper`, `ErrorContext`).
