= nifi-cuioss-rest-processors

`RestApiGatewayProcessor` -- self-contained REST API gateway embedding Jetty 12 with route-based dispatching, per-route JWT authorization, and RFC 9457 error responses.

See link:../doc/architecture/gateway.adoc[Gateway Architecture] for the request pipeline, auth layers, and metrics, and link:../doc/reference/configuration.adoc[Configuration Reference] for route and property details.
